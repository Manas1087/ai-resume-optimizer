🚀 AI Resume Optimizer (Gemini + ATS Friendly)

An AI-powered web application that analyzes job descriptions and resumes to generate optimized, ATS-friendly resumes tailored for specific roles.
Built using Streamlit and Google Gemini (GenAI SDK), this tool helps job seekers improve their resume-job match score and increase shortlisting chances.

🔥 Features

📄 Resume Upload & Parsing

Upload resume in PDF format.
Automatically extracts and processes text.

📊 Job Description Matching

Match Score (%).
Matching Skills.
Missing Keywords.
Suggestions for improvement.

🔘 User-Controlled Workflow

Analyze resume first.
Proceed with optimization manually.

✨ AI Resume Optimization

Adds missing keywords.
Improves bullet points.
Uses strong action verbs.
Maintains ATS-friendly structure.

📥 Download Optimized Resume

Export as PDF / DOCX.
Clean and structured formatting.

🧠 Tech Stack

Frontend: Streamlit.
Backend: Python.
LLM: Google Gemini API (google-genai).
PDF Processing: PyMuPDF.
Document Generation: python-docx, docx2pdf.
